# monikajassova.github.io

My personal blog powered by [Zola static site generator](https://www.getzola.org/) and using [Apollo theme](https://github.com/not-matthias/apollo).

## Getting Started

### 1. Install Zola or download Zola binary and put it into PATH

### 2. Build the site and serve it locally

```
zola build
zola serve
```

### 3. Run check for broken internal and external links

```
zola check
```
